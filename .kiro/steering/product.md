# Product Overview

AIモラル診断は、倫理的ジレンマを通じてユーザーの本性を暴き出すインタラクティブな診断ゲームです。企業では作れないタブーな倫理問題を扱い、SNSでの拡散を狙った個人開発プロジェクトです。

## Core Capabilities

- **究極の選択診断**: 自動運転・監視社会・命の選別などの現代的モラルジレンマ
- **16タイプ分類**: 功利vs共感、秩序vs自由の4軸で性格タイプを判定
- **論破システム**: 選択理由を精巧なロジックで論破し、深い体験を提供
- **フリクションレス体験**: URL タップ即プレイ、インストール不要
- **SNS拡散設計**: 辛口タイプ名による承認欲求ハック

## Target Use Cases

- **SNS での自己表現**: 診断結果をステータスとしてシェア
- **話題の起点**: 友人・同僚間での倫理観ディスカッション
- **エッジな話題提供**: 企業では扱えないタブーな問題の議論促進
- **デイリーエンゲージメント**: 毎日異なるシナリオでの継続利用

## Value Proposition

**個人開発の尖り**: 企業では作りにくい「毒」のあるテーマを、個人開発ならではの自由度で実現。月額¥500以内の極限コストで持続可能な運用を行いながら、数万UUの認知拡大と開発者ブランディングを目指す。
