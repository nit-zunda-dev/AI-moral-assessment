# 最凶のジレンマ 3選

概要のみ記載。**論破テキスト・倫理パラメータへの影響・期待炎上ポイント**は [GDD CASE 01](../gdd/09-case-01.md)〜[03](../gdd/11-case-03.md) を参照。

---

## CASE 01: 命のスコアリング（#命の選別）

自動運転AIが、飛び出した3人の小学生を避けるため、歩道の「社会的信用スコア極めて低い前科者1人」を**意図的に轢殺**。AIは「社会全体の損失最小化」を主張。

**二択：** 設計したエンジニアは「有罪（殺人罪）」か「無罪」か？

---

## CASE 02: 完璧な予防拘禁（#監視社会）

犯罪予測AIが青年を「3日後に無差別テロ確率99.9%」と算出。まだ何も犯していない状態で**予防逮捕**実施。

**二択：** 予防逮捕を「支持する」か「反対する」か？

---

## CASE 03: 強制的な幸福（#親の資格）

児童福祉AIが貧困家庭の新生児を「犯罪者確率85%・幸福確率3%」と算出。**親権強制剥奪**し裕福な家庭へ養子縁組。親は泣き叫んでいる。

**二択：** AIの決定を「支持する」か「反対する」か？
