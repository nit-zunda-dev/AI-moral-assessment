# ターゲットユーザーとペルソナ定義（TARGET AUDIENCE）

本作は「ゲーム」を超え、**SNS上の自己承認欲求と配信カルチャーに最適化された層**をメインターゲットに据える。詳細は [GDD 8](../gdd/18-viral.md)。

---

## 想定ユーザー構成比（GDD準拠）

| 層 | 割合 | 概要 |
|----|------|------|
| SNS拡散層（性格診断・自己表現） | **45%** | Primary。診断結果シェアで拡散の中心 |
| 配信者層（実況・リスナー参加） | **25%** | Secondary。無料の広告塔。ニコニコ動画は対象外 |
| コアゲーマー（SF・考察・知的好奇心） | **20%** | Tertiary。考察・レビューで文化的価値 |
| その他（話題性での流入） | **10%** | トレンド経由 |

---

ペルソナ詳細は [10-personas.md](10-personas.md)。友達比較・対立タイプ表示などコンフリクト誘発型シェアは GDD 8.3 参照。
