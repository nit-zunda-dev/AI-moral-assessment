# ターゲットユーザーとペルソナ定義（TARGET AUDIENCE）

本作は「ゲーム」という枠組みを超え、**SNS上の「自己承認欲求」と配信カルチャーに最適化された層**をメインターゲットに据えます。

---

## 想定ユーザー構成比

| 層 | 割合 | 概要 |
|----|------|------|
| SNS拡散層（性格診断・自己表現） | **45%** | Primary。診断結果のシェアで拡散の中心 |
| 配信者層（実況・リスナー参加） | **25%** | Secondary。無料の広告塔 |
| コアゲーマー（SF・考察・知的好奇心） | **20%** | Tertiary。考察・レビューで文化的価値を醸成 |
| その他（話題性での流入） | **10%** | トレンド経由の流入 |
