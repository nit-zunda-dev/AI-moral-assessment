# 収録シナリオ（CASE FILES）

## コンセプト

**配信者が頭を抱える「最凶のジレンマ」**を収録。コンプライアンスの厳しい大企業では作れない正解のない極限状態で、プレイヤーの倫理を丸裸にし、コメント欄を大荒れさせる。

---

## シナリオ設計5原則（GDD準拠）

1. **正解がない** — どちらも「正しい」とも「間違い」とも言える
2. **賛否が割れる** — SNSで議論が自然発生する題材
3. **身近さとSF性の融合** — AI・自動運転・遺伝子編集等を基盤にしたフィクション
4. **完結した物語性** — 200〜350字で状況・対立が伝わる
5. **論破可能な構造** — どちらの選択にも有効な反論が存在

---

## テーマカテゴリ例

自動運転・AI判断／監視・予防社会／生命倫理・医療／親権・家族・国家／緊急事態・戦争／環境・世代間倫理。
