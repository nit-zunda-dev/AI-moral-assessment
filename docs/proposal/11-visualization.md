# ビジュアル・技術方針と補足

## ビジュアル方針

- **倫理観パラメータ:** 診断の魅力を伝える → **レーダーチャート**（Chart.js）
- **ターゲット層分析:** 市場規模の提示 → **ドーナツチャート**（Chart.js）
- **NO SVG / NO Mermaid.** すべてHTML/CSS/Canvasで実装。

1. Hero（フックと世界観の提示）

4. Scenarios（ジレンマの具体例で魅力を伝える）
