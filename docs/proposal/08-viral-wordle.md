# 戦略2:「Wordle型」日替わりトレンドジャック

大ヒットブラウザゲーム『Wordle』や『Suika Game』の手法を踏襲する。

---

## 仕組み

- 全プレイヤーに**「毎日同じ1つのお題」**を出題する。
- 結果だけを**定型文でシェア**させる。
- 毎日のように**SNSのトレンド入り**を狙う。

---

## 日替わりシェアフォーマット例

```
今日の『AIモラル診断』 #045
CASE: 強制的な幸福

私の決断：【親の権利を剥奪する】
AIの評価：冷酷度 98% 💀

「あなたは親権よりも効率を選んだ」
https://aimoral.example.com/daily/045
#AIモラル診断
```

※ネタバレを防ぎつつ「お前はどう判断した？」と議論を誘発するフォーマット。

---

## 効果

- **自然発生的な議論:** 全員が「同じお題」に挑むため、タイムライン上で議論が毎日生まれる。
- **圧倒的な継続率:** 毎日深夜0時にお題を更新。LLMによるニュース記事からの自動生成と組み合わせれば、運用コストゼロで毎日の「習慣」として定着させられる。
