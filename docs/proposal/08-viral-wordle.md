# 戦略2:「Wordle型」日替わりトレンドジャック

全プレイヤーが**同じ1問**に挑む → SNSで「今日のCASE、○○が正解だろ」が毎日発生 → ネタバレなしシェア → 習慣化。詳細は [GDD 8.2](../gdd/18-viral.md)。

---

## 仕組み

- 毎日0:00 JSTに1問配信（デイリーモード）。1日1回のみプレイ可（Cookie・DBなし）。
- 翌日テーマは前日23:00にティーザー公開。
- 結果を**定型文でシェア**させ、トレンド入りを狙う。

---

## シェアテキスト例（GDD準拠）

```
今日の #AIモラル診断 #045
CASE: 強制的な幸福

私の判断：【親の権利を剥奪する】
AIの論破：「あなたは...」

診断タイプ: Type05「善意の独裁者」
https://aimoral.pages.dev/daily/045
```

※ネタバレを防ぎつつ「お前はどう判断した？」と議論を誘発。

---

## 効果

- **自然発生的な議論** — 全員が同一お題のためタイムラインで毎日議論。
- **継続率** — 運用コストゼロで「習慣」として定着（将来: ニュース記事からのお題自動生成も検討）。
