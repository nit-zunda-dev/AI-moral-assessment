# Core Gameplay Loop

単なるアンケートではありません。**精巧なシナリオと診断ロジックが、プレイヤーの「倫理」を6軸でスコアリングし、16タイプのどれかに分類します。**（体験は「AIが論破している」ように見せる事前生成型）

---

## ステップ概要（GDD準拠）

1. **ジレンマ提示** — 正解のない残酷な二択の事案を表示（専用イラスト付き）。
2. **二択選択** — A/Bの大きなボタン（スワイプ可）。「中立」は存在しない。
3. **論破演出** — 選択後、判決エフェクト→論破テキストをタイピング表示（事前生成テキスト）。
4. **次へ** — 「異議あり」「受け入れる」の2択（どちらも次へ進む演出）。
5. **診断レポート** — 16タイプ判定 + 6軸レーダーチャート + 辛口コメント。
6. **SNSシェア** — ネタバレなし定型文・OGP動的生成。

---

## ゲームモード

- **デイリーモード:** 全プレイヤーが同一1問。毎日0:00 JST更新。1日1回のみ（Cookie管理）。
- **シリーズモード:** テーマ別5〜10問パック。
- **ランダムモード:** 過去ケースからランダム5問。

**倫理観パラメータ（6軸）:** 結果・効率、ルール・規律、人情・道徳、自己保身、共感・優しさ、論理・冷酷さ。詳細は [GDD ゲームシステム](../gdd/03-game-system.md)。
