# 認知獲得のためのバズ戦略（USER ACQUISITION STRATEGY）

広告費ゼロでユーザーを**雪だるま式に増やす**ための、ゲームデザインに組み込まれた2つの強力な拡散装置です。

---

## 戦略1:「残酷なまでの」自己分析シェア

- 専門用語を排除し、**MBTIのように「直感的に分かりやすい性格タイプ」**として結果を出力する。
- **思わずツッコミを入れたくなる辛口なキャッチコピー**で、SNSでの拡散（承認欲求と自己表現）を強烈に促す。

### 出力例

- **倫理観タイプ:** 冷酷無比な「計算機」
- **説明例:** 「多数を救うためなら少数を切り捨てる」。あなたは感情を完全に排除し、徹底的に効率と結果を求めるタイプ。AIよりもAIらしいかもしれません。
- **相性の良い相手:** 従順なロボット
- **AIからの評価:** 「サイコパス気質」

### シェア導線

「結果をシェアして反論を待つ」など、SNS共有を前提にしたUIを配置する。
