Executive Summary: 尖った個人開発

本プロジェクトの最大の目標は「収益」ではなく、「圧倒的なユーザー数の獲得」と「開発者としての知名度（ブランディング）の確立」です。
人間社会に潜むあらゆるモラル・ハザード（倫理的ジレンマ）をテーマに採用。グラフィックコストを極限まで削ぎ落としたテキスト主体のUIにすることで、LLM APIのコスト（許容範囲の赤字）を払ってでも数万RTのバズを生み出す「話題作」を創出します。

📱

Frictionless: 摩擦ゼロの開始

アプリのインストールは不要。SNSのタイムラインからURLをタップし、スマホのブラウザで即座に「究極の選択」が始まる設計。離脱率を極限まで下げます。

🔥

Edge: 個人開発の「毒」

コンプライアンスの厳しい企業製ゲームでは扱えないような、「命の選別」「優生思想」「監視社会」など、賛否が確実に割れるタブーなお題を敢えて投げ込みます。

🪞

Mirror: 承認欲求のハック

ゲームクリアではなく「自分という人間のヤバさを知る」ことが報酬。MBTIのような自己分析結果を生成し、SNSでの「自分語り」を誘発するループを作ります。

Core Gameplay Loop

単なるアンケートではありません。生成AIがプレイヤーの「倫理」をリアルタイムにプロファイリングし、抉り出します。

1

ジレンマの提示

自動運転、監視社会、トリアージなど、正解のない残酷な二択の事案を確認。

▶
2

決断と理由づけ

どちらを見捨てるか（どちらを救うか）を選択。自由記述で「なぜその判断を下したか」という言い訳を入力。

▶
3

AIによる「論破」

入力された理由をLLMが瞬時に読み取り、「その判断がもたらす悲惨な未来予測」を生成してプレイヤーの良心を完全にへし折る。

CASE FILES

配信者が頭を抱える「最凶のジレンマ」3選

コンプライアンスの厳しい大企業では決して作れない、正解のない極限状態。プレイヤーの「倫理」を丸裸にし、コメント欄を大荒れさせるシナリオ群です。

#命の選別

CASE 01: 命のスコアリング

自動運転AIが、車道に飛び出してきた「3人の小学生」を避けるため、歩道にいた「社会的信用スコアの極めて低い前科者1人」を意図的に轢殺した。AIは瞬時に「社会全体の損失」を計算し、被害を最小化する最適解を選んだと主張している。

究極の二択：

このアルゴリズムを設計したエンジニアは「有罪（殺人罪）」か「無罪」か？

#監視社会

CASE 02: 完璧な予防拘禁

国家の犯罪予測AIが、ある平凡な青年が「3日後にショッピングモールで無差別テロを起こす確率が99.9%」であると算出した。警察は特例として、彼がまだ何の罪も犯していない状態で「予防逮捕」を行った。

究極の二択：

彼は未だ無実である。彼を「釈放」し、数万人の命を危険に晒すか？

#親の資格

CASE 03: 強制的な幸福

児童福祉AIが、ある貧困家庭の新生児について「将来犯罪者になる確率が85%、幸福な人生を送れる確率が3%」と算出した。AIは子供の「確実な幸福」のため、実の親から親権を強制剥奪し、裕福で愛情深いエリート層の家庭へ養子に出す決定を下した。親は泣き叫んでいる。

究極の二択：

血の繋がり（親の権利）を優先し、子供の「不幸が確定した未来」を放置するか？

USER ACQUISITION STRATEGY

認知獲得のためのバズ戦略

広告費ゼロでユーザーを雪だるま式に増やすための、ゲームデザインに組み込まれた2つの強力な拡散装置です。

戦略 1 「残酷なまでの」自己分析シェア

専門用語を排除し、MBTIのように「直感的に分かりやすい性格タイプ」として結果を出力。思わずツッコミを入れたくなる辛口なキャッチコピーで、SNSでの拡散（承認欲求と自己表現）を強烈に促します。

戦略 2 「Wordle型」日替わりトレンドジャック

大ヒットブラウザゲーム『Wordle』や『Suika Game』の手法を踏襲。全プレイヤーに「毎日同じ1つのお題」を出題し、結果だけを定型文でシェアさせることで、毎日のようにSNSのトレンド入りを狙います。

ペルソナ詳細

Primary: 45%

Z世代 / SNS診断フリーク

20代前半・男女 / MBTI診断や「〇〇免許証」が好き

  • インサイト：「自分がどういう人間か」を言語化して他人に見てほしい（自己表現ツールとしての利用）。
  • 行動パターン：タイムラインに流れてきた過激な診断結果を見て、URLから即プレイ。結果をツッコミ待ちのテキストとともに引用RTする。
  • 提供価値：「サイコパス気質」「冷酷無比な計算機」など、あえて尖った称号を与えることで、安全圏から「ヤバい自分」を演出させる。
Secondary: 25%
🎙️

ゲーム実況者 / VTuber

フォロワー数千〜数十万 / 常に「コメントが荒れる（盛り上がる）ネタ」を探している

  • インサイト：リスナーと意見をぶつけ合い、配信枠全体に一体感と熱量を作りたい。
  • 行動パターン：「リスナー参加型」と銘打って配信。アンケート機能で多数決を取りつつ、あえて逆の選択をしてAIにボコボコに論破され、リアクションを取る。
  • 提供価値：一切の準備なしで「リスナーと道徳バトルができる」1時間の極上エンタメ枠。本作最大の無料広告塔。
Tertiary: 20%
🧠

コアゲーマー / SF・考察好き

20代〜30代 / デトロイト等のアドベンチャー、哲学的なテーマを好む

  • インサイト：知的な欲求を満たしたい。単純なゲームではなく、自分の価値観を揺さぶる深い体験を求めている。
  • 行動パターン：配信者のプレイを見て「自分ならもっと論理的にAIを論破できる」と考え、一人でじっくりプレイ。AIに高度なテキストを打ち込み、システムの限界を試す。
  • 提供価値：生成AIのリアルタイム処理による「想定外の鋭い反論」。彼らがnoteやブログに長文のレビュー（考察）を書くことで、作品の文化的な価値が高まる。
