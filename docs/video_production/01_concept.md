# 動画コンセプト・キャラクター設定

## 動画コンセプト

### 基本方針

「答えのない問いを、一緒に考える」

各シナリオの解説動画は、正解を提示するのではなく、**視聴者が自分の判断を問い直すきっかけ**を与えることを目的とします。ずんだもんが「素朴な疑問を持つ側」、四国めたんが「深い哲学的視点を持つ側」という役割分担で対話形式の解説を行います。

### 動画が目指す感情体験

1. **「あれ、私の答えって本当に正しかったの？」** という気づき
2. **「どっちの立場も、なんかわかる……」** という葛藤
3. **「もう一度考えてみよう」** という行動促進

動画を見終わったあと、視聴者が「考えさせられた」と感じることが最大の成功指標です。

| 項目 | 設定 |
|---|---|
| 役割 | 進行役・素朴な疑問担当 |
| 一人称 | ぼく |
| 語尾 | 〜なのだ、〜のだ |
| キャラ像 | 善意で物事を考えるが、深く掘り下げると迷う純粋な視点を持つ |
| 担当テーマ | ユーザーの選択の「気持ちに寄り添う側」の立場を代弁 |

**例文：**
> 「でも、あの子供のことを考えたら……仕方なかったと思うのだ。」
> 「ぼくなら、きっとそっちを選んでたのだ。うーん、でも……。」

## 動画の基本構成（各エピソード共通）

```
[PART 1] ケース提示 (約1分)
  ずんだもんがシナリオのあらすじを紹介
  「今日のケースはこれなのだ！」

[PART 2] 選択肢の解説（約1〜1.5分）
  選択A・Bの主張をそれぞれ提示
  両方の「もっともな理由」を丁寧に説明

[PART 3] 反論の深堀り（約1.5〜2分）※メインパート
  選択Aへの反論、選択Bへの反論を掘り下げ
  「でもこう考えると…」という形で深める
  感情に訴える反論例と、論理的な反論例の両方を提示

[PART 4] 哲学的背景（約1分）
  このシナリオに関連する倫理学の概念・思想家を紹介
  「功利主義」「義務論」「徳倫理学」など

[PART 5] まとめ・問いかけ（約30〜45秒）
  「正解はないかもしれないけど、考え続けることに意味がある」
  コメント欄・SNSへの誘導
```

**目標動画尺：5〜7分**

---

## トーン・スタイルガイドライン

- **シリアスになりすぎない**。重いテーマでも、対話のテンポは軽快に保つ
- **視聴者を責めない**。どちらの選択も「それはそれで理解できる」というスタンス
- **難しい言葉を避ける**。「功利主義」などを使う場合は必ず噛み砕いた説明を添える
- **BGMは抑えめ**。思考を邪魔しないアンビエント系を推奨
- **立ち絵の演技**。ずんだもんは驚き・困惑・共感の感情表現を多用。めたんは落ち着いた姿勢から徐々に真剣な表情へ

## アプリとの連携

解説動画はアプリのフィードバック画面に埋め込まれます。

```
[判断結果画面]
    ├── AIフィードバックテキスト（上部）
    └── 解説動画（下部）← YouTube埋め込み or 直接再生
```

動画はYouTubeにアップロードし、各シナリオJSONの `videoUrl` フィールドから参照する形を想定します。
