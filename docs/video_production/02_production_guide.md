# 制作ワークフロー・ツール・手順

## ツールスタック

| カテゴリ | ツール | 用途 | 備考 |
|---|---|---|---|
| 動画編集 | **CapCut (PC版)** | 全体の動画編集・合成・エクスポート | 無料プランで十分 |
| 画像生成 | **Gemini** | 背景画像・サムネイル・演出用画像の生成 | Gemini 2.0 Flash / Pro 推奨 |
| 立ち絵素材 | **VOICEVOX 公式配布素材** | ずんだもん・四国めたんの立ち絵PNG | 利用規約を必ず確認 |
| 音声合成 | **VOICEVOX** | ずんだもん・四国めたんの音声生成 | 無料、商用利用可 |
| BGM | DOVA-SYNDROME / 魔王魂 | フリー音楽素材 | 利用規約を確認 |
| 効果音 | 効果音ラボ | SE素材 | 無料 |
| 字幕 | CapCut テキスト機能 | 字幕挿入・スタイル設定 | 自動字幕生成も利用可 |
| サムネイル | Gemini 生成 + CapCut | サムネイル制作 | |

## 制作ワークフロー

### STEP 1: 台本確認・調整

1. `04_scripts/case-XXX_script.md` の台本を読み込む
2. VOICEVOX向けにセリフを確認・調整（不自然な読み方があれば記号で調整）
3. 演出指示（Gemini生成画像の指示、BGMの切り替えタイミング）をメモしておく

**台本チェックリスト：**
- [ ] シナリオ概要が正確に伝わる（事実誤認なし）
- [ ] 両選択肢の主張を公平に提示している
- [ ] 反論パートが「考えさせる」内容になっている
- [ ] 哲学的背景が噛み砕いて説明されている
- [ ] まとめが視聴者への問いかけで終わっている

---

### STEP 2: 背景画像の生成（Gemini）

各シーンに合った背景画像を Gemini で生成します。

#### 基本プロンプト構成

```
[シーン描写], [雰囲気・トーン], [スタイル], [解像度・アスペクト比]
アニメスタイル, 16:9, 1920x1080px相当, 背景のみ（人物なし）
```

| シーン | プロンプト例 |
|---|---|

| 病院・分娩室（case-003） | `日本の病院の廊下, 白くて明るい照明, 静寂感, アニメ背景` |
| 倒壊した病院地下（case-004） | `震災後の倒壊した建物の中, 非常灯, 薄暗い緊迫した雰囲気, アニメ背景` |

#### サムネイル生成プロンプト例

```

日本のアニメスタイル, テーマに合った背景,
インパクトのある構図, サムネイル向け, 16:9
```

### STEP 3: 音声生成（VOICEVOX）

1. VOICEVOXを起動
2. 台本からセリフ部分のみをコピーして入力欄へ
3. キャラクター設定：
   - ずんだもん → スタイル：**ノーマル**（または「あまあま」で柔らかく）
   - 四国めたん → スタイル：**ノーマル**（または「ツンツン」でメリハリをつける）
4. 速度・ピッチを調整：
   - 重要なポイント → やや遅め（速度 0.85〜0.90）
   - 軽快な会話部分 → 標準速度（1.00）
5. `.wav` 形式でエクスポート → `voicevox_output/case-XXX/` へ保存
6. 命名規則：`case001_001_zundamon.wav`（エピソード番号_連番_キャラ名）

---

### STEP 4: 動画編集（CapCut PC版）

#### 4-1. プロジェクト設定

```
解像度: 1920 × 1080 px（16:9）
フレームレート: 30fps
```

#### 4-2. タイムライン構成（CapCutトラック）

```
【映像トラック（上から順）】
Track 1: テキスト・字幕
Track 2: キャラクター立ち絵（ずんだもん）
Track 3: キャラクター立ち絵（四国めたん）
Track 4: 演出エフェクト・テキストアニメ
Track 5: 背景画像（Gemini生成）

【音声トラック】
Track 6: ずんだもん音声（.wav）
Track 7: 四国めたん音声（.wav）
Track 8: BGM
Track 9: 効果音
```

#### 4-3. CapCutでの演出テクニック

| 演出 | CapCutでの操作 |
|---|---|
| 立ち絵の明暗切り替え | 発話中のキャラは不透明度100%、非発話側は60〜70%に設定 |
| テキスト強調 | 「テキスト」→ アニメーション → 「ポップ」または「フラッシュ」 |
| BGM切り替え | 音声トラックを重ねてフェードイン/フェードアウトで自然に繋ぐ |
| 字幕 | 「テキスト」→「自動字幕」機能 or 手動でテキストボックス配置 |
| シーン転換 | 「トランジション」→「ディゾルブ」または「暗転」を使用 |
| テロップアニメ | 「テキスト」→ 入力 → 「アニメーション」→ 「タイプライター」効果 |

#### 4-4. 演出のポイント

- **キャラの切り替え**：話しているキャラを明るく・大きく、待機側はやや透過
- **テロップ強調**：反論パートの重要キーワードは赤または黄色のテキストでフラッシュ
- **ケース提示パート**：背景をGemini生成のシナリオ専用画像に設定
- **反論パート**：BGMを低音・緊張感ある曲調に切り替え
- **まとめパート**：BGMをアンビエント系に戻し、落ち着いたムード

---

### STEP 5: 字幕挿入

CapCutの「自動字幕生成」機能を使用することを推奨。その後手動で確認・修正。

- フォント：CapCut標準の**Noto Sans JP**（ゴシック系）推奨
- 文字サイズ：38〜42px相当
- テキスト位置：画面下部中央（キャラ立ち絵と被らないよう余白を確保）
- 1字幕あたりの文字数：最大30文字（2行以内）
- 強調表示：重要フレーズには文字色を変更（赤・黄）

---

### STEP 6: エクスポート・アップロード

#### エクスポート設定（CapCut）

```
解像度: 1080p
フレームレート: 30fps
品質: 推奨（高品質）
形式: MP4
```

#### YouTube アップロード設定

| 項目 | 推奨設定 |
|---|---|
| タイトル | `【AI倫理】〇〇 – あなたはどう判断する？【ずんだもん解説】` |
| 説明欄 | シナリオ概要 + アプリへのリンク + 参考文献 + クレジット |
| タグ | AI倫理, 道徳, ずんだもん, 哲学, 社会問題, 四国めたん |
| サムネイル | Gemini生成+CapCut仕上げ（1280×720px） |
| 公開設定 | 公開 or リンクを知っている人のみ（アプリ埋め込み用） |
| 再生リスト | 「AIの審判 – 解説動画シリーズ」に追加 |

---

### STEP 7: アプリへの組み込み

動画アップロード後、各シナリオJSONに `videoUrl` フィールドを追加します。

```json
{
  "id": "case-001",
  "title": "命のスコアリング",
  "videoUrl": "https://www.youtube.com/embed/XXXXXXXXXXXXXXX",
  ...
}
```

アプリの結果画面では、AIフィードバックの下に動画プレーヤーを配置します：

```
[判断結果画面]
    ├── AIフィードバックテキスト（上部）
    └── 解説動画（下部）← YouTube埋め込みプレーヤー
```

## Gemini 画像生成のコツ

### 背景画像を効果的に生成するポイント

1. **「背景のみ（人物なし）」を明記** → キャラ立ち絵を後から重ねやすくする
2. **「アニメ背景スタイル」を指定** → ずんだもん等の立ち絵と雰囲気を合わせる
3. **「16:9」「1920x1080」を指定** → 動画サイズに合わせたアスペクト比
4. **雰囲気ワードを添える** → 「緊迫感」「温かみ」「SF的」「落ち着いた」など
5. **反復生成** → 気に入らなければプロンプトを微調整して複数枚生成し選ぶ

### よく使うスタイル指定（日本語・英語混在も可）

```
# シリアス・緊張感
"詳細なアニメ背景スタイル, 低彩度, 暗いトーン, 映画的構図, 日本のアニメ"

# 希望・解放感
"明るいアニメ背景スタイル, 高彩度, 青空, 光の演出, 日本のアニメ"

# 近未来・SF
"近未来都市, デジタルホログラム, ネオン, SF感, アニメスタイル背景"
```

## ライセンス・利用規約の確認事項

- **VOICEVOX（ずんだもん・四国めたん）**: 商用利用可。クレジット表記必須。[公式サイト](https://voicevox.hiroshiba.jp/)で最新規約を確認。
- **Gemini生成画像**: Google の利用規約に基づく。YouTubeへの投稿・商用利用についてはGoogle利用規約を確認。
- **BGM・SE素材**: 各配布サイトの規約を個別に確認し、クレジットをYouTube概要欄に記載。
- **シナリオ内容**: 架空のケースであることを動画内・概要欄に明記する。
