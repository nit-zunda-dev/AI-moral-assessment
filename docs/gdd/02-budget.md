# 2. 予算・コスト設計

## 2.1 月間コスト上限 ¥500 の実現方針

**最大の課題は「AI APIコスト」**。GPT-4o-miniでも月10,000プレイで約$3〜5かかり、バズ時には即座に予算超過する。そのため**AIをリアルタイム呼び出ししない設計**を採用する。

- **NG設計:** プレイヤーの回答 → リアルタイムAPI呼び出し → 個別生成（10,000プレイ × $0.005 = $50 → 予算オーバー）
- **OK設計:** 事前に全パターンの「論破テキスト」を生成してJSONとして格納。開発時に一度だけAPIを呼ぶ = $0〜$5（開発費として許容）

---

## 2.2 採用アーキテクチャ「静的コンテンツ + クライアントサイド診断」

リアルタイムAIを使わず、精巧に設計されたシナリオツリーと診断ロジックで「AIっぽい体験」を実現する。

| 項目 | 費用 |
|------|------|
| Cloudflare Pages（静的ホスティング） | **¥0** |
| Cloudflare Workers（SSR/API・100,000 req/day無料枠） | **¥0** |
| シナリオJSON（静的ファイル） | **¥0** |
| 診断ロジック（TypeScript/クライアント） | **¥0** |
| ドメイン（.pages.dev サブドメイン使用） | **¥0** |
| カスタムドメイン（任意・年払い） | **¥150/月** |
| シナリオAI生成（開発時のみ一度） | **¥500〜¥1,500**（初期投資のみ）|
| **合計（月額）** | **¥0〜¥150** ✅ |

---

## 2.3 事前生成型AI体験

プレイヤーは「AIがリアルタイムに論破している」と感じるが、実際は**開発時にあらかじめ全パターンを生成したJSONを返している**。将来拡張: トラフィック安定後に Cloudflare AI Workers (Llama 3, Mistral) のリアルタイム生成へ移行可能（無料枠 10,000 neurons/day）。
