# 4.2 全16タイプ詳細（Type 01〜08）

#### Type 01「冷酷な設計者」UEOA
**英名:** The Architect of Sacrifice  
**キャッチ:** 「あなたはシステムで人を殺せる人間です」  
特徴: 規律に従いながら最大多数の幸福を計算。感情を排除した功利主義者。強み=論理的一貫性・決断力。弱み=少数者への無感覚。相性悪=Type 09。

#### Type 02「システムの番人」UEOI
**キャッチ:** 「ルールが間違っていても、あなたはルールを守り続けます」  
特徴: 規律と効率を同時に重視、個人の権利には鈍感。強み=実行力・組織への忠誠。弱み=権威への盲目的服従。相性悪=Type 13。

#### Type 03「臆病な計算機」UEPA
**キャッチ:** 「あなたは正しい答えを知りながら、行動しない人間です」  
特徴: 功利的正解を頭では理解しているが責任を恐れて傍観。強み=状況分析力。弱み=決断力の欠如。相性悪=Type 05。

#### Type 04「秩序の奴隷」UEPI
**キャッチ:** 「あなたが盲目的に守るルールは、誰かが作ったものです」  
特徴: ルールを疑わず服従、行動力もない指示待ちの最終形態。強み=規則遵守。弱み=自己意思の欠如。相性悪=Type 16。

#### Type 05「善意の独裁者」UFOA
**キャッチ:** 「あなたの優しさは、他者の自由を奪うことで成立しています」  
特徴: 「みんなのため」を口実に他者の選択を侵食。強み=実行力・ビジョン。弱み=独善性。相性悪=Type 12。

#### Type 06「利己的な合理主義者」UFOI
**キャッチ:** 「あなたの論理は完璧ですが、それはあなただけを救う論理です」  
特徴: 自己利益と社会効率を同一視。強み=自己管理・論理。弱み=共感能力の低さ。相性悪=Type 09。

#### Type 07「沈黙する傍観者」UFPA
**キャッチ:** 「あなたは関わらないことで、最も多くの人を傷つけます」  
特徴: 「誰かがやるだろう」の傍観者効果の体現。強み=状況認識。弱み=行動力欠如。相性悪=Type 05。

#### Type 08「哲学的ニヒリスト」UFPI
**キャッチ:** 「あなたはすべてを疑いすぎて、何も選べなくなっています」  
特徴: あらゆる倫理命題に懐疑的で虚無に至る。強み=深い思考力。弱み=行動不能。相性悪=Type 01。
