# 11. KPIと成功指標・付録

## 11.1 定量KPI

| 指標 | 目標（リリース後）|
|------|------------------|
| 累計プレイ（1ヶ月後） | 10,000回以上 |
| 累計プレイ（3ヶ月後） | 100,000回以上 |
| SNSシェア率（結果画面） | 20%以上 |
| デイリー翌日リテンション | 15%以上 |
| Xトレンド入り | 30日以内に1回 |
| 配信者プレイ | フォロワー1万以上が5名以上 |
| **月間サーバーコスト** | **¥500以内（厳守）** |

## 11.2 定性的成功基準

- Xで「今日のCASEは〇〇が正解だろ」という議論が毎日観測できる
- note・ブログに本作の考察記事が5件以上自然発生
- 開発者個人のSNSフォロワーがリリース前比300%以上増加

## 付録: 用語集

| 用語 | 定義 |
|------|------|
| 事前生成型AI体験 | 開発時のみAPI呼び出しでレスポンスを静的JSON化、本番ではリアルタイム呼び出しなし |
| 倫理観パラメータ | 6軸でスコアリングした値 |
| 16タイプ分類 | 4つの二元軸から導出する性格タイプ（MBTI構造準拠）|
| デイリーモード | 毎日0:00 JSTに1問配信するメインモード |
| OGP動的生成 | Workersでシェアカード画像をリクエスト時に生成する仕組み |
| Wordle型フォーマット | 全員が同日に同一問題を解き、ネタバレなし定型文でシェアするUX |
