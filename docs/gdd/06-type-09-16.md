# 4.2 全16タイプ詳細（Type 09〜16）

#### Type 09「泣き叫ぶ聖者」EEOA
**キャッチ:** 「あなたの涙は美しいですが、涙は誰も救いません」  
特徴: 弱者への共感が極めて強く感情的。強み=共感・道徳的勇気。弱み=感情的判断。相性悪=Type 01。

#### Type 10「正義の執行者」EEOI
**キャッチ:** 「あなたの正義は強く美しいですが、それはあなたの正義です」  
特徴: 個人の権利と正義感を規律の中で実行。共感と論理のバランスが良い。強み=バランス。弱み=正義の押し付け。相性悪=Type 08。

#### Type 11「殉教者」EEPA
**キャッチ:** 「あなたはすべての人を愛しすぎて、誰も救えません」  
特徴: 全員を守ろうとして全員を守れない。強み=道徳的純粋さ。弱み=決断力欠如。相性悪=Type 06。

#### Type 12「孤独な理想主義者」EEPI
**キャッチ:** 「あなたの理想は正しい。ただし、この世界では実現しません」  
特徴: 理想の倫理観はあるが現実に適用する力がない。強み=倫理的洞察。弱み=実行力。相性悪=Type 05。

#### Type 13「反乱する詩人」EFOA
**キャッチ:** 「あなたの反抗は美しいですが、それだけでは何も変わりません」  
特徴: 体制・権威に本能的に反発。弱者への共感から行動するが代替案を持たない場合が多い。相性悪=Type 02。

#### Type 14「自由の守護者」EFOI
**キャッチ:** 「あなたが守る自由は、時として他者の安全を脅かします」  
特徴: 個人の自由と共感を高い次元で両立。最もバランスの取れた共感派。相性悪=Type 04。

#### Type 15「優しい傍観者」EFPA
**キャッチ:** 「あなたは最も優しい人間ですが、最も役に立たない人間です」  
特徴: 全員に共感し誰も傷つけたくないが行動できず。強み=共感・非暴力。弱み=行動力完全欠如。相性悪=Type 01。

#### Type 16「燃える革命家」EFPI
**キャッチ:** 「あなたは世界を変えたいのに、まだ始めていません」  
特徴: 強烈な共感と自由への欲求を持ちながら未だ行動に移せていない。可能性の塊。相性悪=Type 04。
