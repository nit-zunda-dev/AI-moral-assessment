# 1. ゲーム概要

## 1.1 エレベーターピッチ

> *「自動運転・監視社会・命の選別──正解のない究極の選択があなたの"本性"を暴く。AIではなく、精巧に設計されたシナリオと診断ロジックが、あなたを16の倫理観タイプのどれかに分類する。スマホから即プレイ・即シェア。広告費ゼロで数万RTを狙うブラウザゲーム。」*

---

## 1.2 コアコンセプト 3原則

| 原則 | 内容 |
|------|------|
| **Frictionless** | インストール不要。URLタップ即プレイ。離脱率ゼロを目指す。 |
| **Edge** | 企業では作れないタブーな倫理問題を投入し、議論を誘発する。 |
| **Mirror** | 承認欲求のハック。辛口タイプ名でSNS自己表現を促す。 |

---

## 1.3 プロジェクト最優先目標

本プロジェクトは**収益最大化が目的ではない**。

1. **認知拡大** ─ SNSで数万UU獲得
2. **開発者ブランディング** ─ 個人開発者としての知名度確立

月¥500以内の運用コストで持続可能な状態を維持しながら最大限のバズを生み出す設計を優先する。
