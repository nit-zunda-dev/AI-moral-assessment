# 4.3 タイプ相関マップ

```
       [功利派 U]
          │
  UEOA(01)│UEOI(02)  UFOA(05)  UFOI(06)
 ─────────┼──────────────────────────────
  秩序     │                         自由
 ─────────┼──────────────────────────────
  UEPA(03)│UEPI(04)  UFPA(07)  UFPI(08)
          │
 ─────────┼──────────────────── [共感派 E]
          │
  EEOA(09)│EEOI(10)  EFOA(13)  EFOI(14)
 ─────────┼──────────────────────────────
          │
  EEPA(11)│EEPI(12)  EFPA(15)  EFPI(16)
          │
```

- **U/E:** 功利派 vs 共感派
- **O/F:** 秩序派 vs 自由派
- **A/P:** 能動派 vs 受動派
- **I/C:** 個人派 vs 集団派（図では I を右側に配置）

相性の悪いタイプ同士は対角・隣接で配置されることが多い。診断結果画面で「最も対立するタイプ」を表示し、議論・シェアを促す。
