# 5. シナリオ設計

## 5.1 シナリオ設計5原則

| 原則 | 内容 |
|------|------|
| **① 正解がない** | どちらの選択肢も「正しい」と言えず、どちらも「間違い」と言える |
| **② 賛否が割れる** | SNSで議論が自然発生するよう社会的に二極化する題材を選定 |
| **③ 身近さとSF性の融合** | 現実の技術（AI・自動運転・遺伝子編集）を基盤にしたフィクション |
| **④ 完結した物語性** | 200〜350字で状況・登場人物・対立が完結して伝わる |
| **⑤ 論破可能な構造** | どちらの選択にも有効な反論が存在する設計 |

---

## 5.2 シナリオ全体マップ（テーマカテゴリ）

- **自動運転・AI判断:** 命の選別アルゴリズム、社会的信用スコア、事故責任の帰属
- **監視・予防社会:** 犯罪予測AIと予防逮捕、SNS監視と思想管理、遺伝子犯罪予測
- **生命倫理・医療:** トリアージの選択、安楽死の合法化、遺伝子編集の許容範囲
- **親権・家族・国家:** AI親権剥奪判断、強制的な幸福、子どもの意志 vs 親の権利
- **緊急事態・戦争:** 一人を犠牲に多数を救う、核抑止の倫理、テロリストとの交渉
- **環境・世代間倫理:** 現世代 vs 将来世代、動物の命 vs 人間の便益
