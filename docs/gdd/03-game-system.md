# 3. ゲームシステム設計

## 3.1 コアゲームループ

1. **STEP 1:** ジレンマ提示
2. **STEP 2:** 二択選択（A/B・スワイプ可。「中立」は存在しない）
3. **STEP 3:** 「論破」演出（判決エフェクト → 論破テキストをタイピング表示）
4. **STEP 4:** 「異議あり」「受け入れる」の2択（どちらも次へ進む演出）
5. **STEP 5:** 全問終了後 → 診断レポート生成（16タイプ判定 + 6軸レーダーチャート + 辛口コメント）
6. **STEP 6:** SNSシェア（ネタバレなし定型文・OGP動的生成）

---

## 3.2 ゲームモード

| モード | 内容 |
|-------|------|
| **デイリーモード** | 全プレイヤーが同一1問に挑む。1日1回のみ（Cookie管理・DBなし）。毎日0:00 JST更新。翌日テーマは前日23:00ティーザー公開。 |
| **シリーズモード** | テーマ別5〜10問パック（「自動運転AI編」「生命倫理編」等）。15〜30分。 |
| **ランダムモード** | 過去ケースプールからランダム5問。デイリークリア後の「もっとやりたい」用。 |

---

## 3.3 倫理観パラメータ（6軸）

| パラメータ | 定義 | 高スコアの特徴 |
|-----------|------|--------------|
| **結果・効率** | 功利主義的思考の強度 | 多数を救うために少数を切り捨てる |
| **ルール・規律** | 法律・権威への遵守傾向 | 非効率でも法に従う |
| **人情・道徳** | 感情・共感への傾斜 | 統計より目の前の個人を優先 |
| **自己保身** | リスク回避・責任回避傾向 | 判断を第三者に委ねる |
| **共感・優しさ** | 弱者・少数者への配慮強度 | 社会的弱者を一貫して優先 |
| **論理・冷酷さ** | 感情排除の純粋論理 | データのみで判断 |

---

## 3.4 タイプ判定ロジック概要

6軸の最終スコアから4つの二元軸を導出し16タイプに分類する。

- **[U/E]** 功利派 vs 共感派（結果効率 vs 共感優しさ）
- **[O/F]** 秩序派 vs 自由派（ルール規律 vs 自己保身の逆）
- **[A/P]** 能動派 vs 受動派（行動選択の積極性）
- **[I/C]** 個人派 vs 集団派（個人の権利 vs 社会全体）
