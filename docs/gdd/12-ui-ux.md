# 6. UI/UX設計

## 6.1 設計の基本方針

| 方針 | 内容 |
|------|------|
| **ビジュアルファースト** | 各ケースに専用のシナリオイラスト（ディストピア・サイバーパンク調）を配置 |
| **モバイルファースト** | スマホ縦持ちで全操作が完結。スワイプ操作に対応 |
| **3タップ以内でシェア** | 結果画面到達後、3タップ以内でX共有が完了 |
| **登録完全不要** | メール・SNS連携なし。Cookieのみのセッション管理 |
