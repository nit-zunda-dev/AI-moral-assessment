# 6.3.5 診断結果画面・OGP・アニメーション

## 診断結果画面（シェアの核）

- 「AIモラル診断 結果」
- あなたの倫理観タイプ
- **タイプ専用イラスト**（16タイプそれぞれにキャラクターイラスト）
- Type XX「冷酷な設計者」等・英名・キャッチコピー
- 6軸レーダーチャート（SVG）
- 詳細コメント
- **メインCTA:** 「𝕏 結果をシェアする」
- 友達の結果と比較する

---

## OGP / シェアカード設計

Xのカード表示用OGP画像を **Cloudflare Workers で動的生成**。  
構成: ロゴ・Type XX「冷酷な設計者」・タイプイラスト＋レーダーチャート・キャッチコピー・URL。  
技術: Workers + `@resvg/resvg-wasm` で SVG→PNG をエッジ生成。Cache-Control 24時間。

## アニメーション仕様

| 要素 | アニメーション | 実装 |
|------|--------------|------|
| ケース遷移 | スライドイン（左→右） | CSS Transitions |
| 二択ボタン | ホバーでスケール+シャドウ | CSS Transform |
| AI論破テキスト | タイピング | TypeScript setInterval |
| レーダーチャート | 0→最終値の描画 | SVG SMIL / CSS |
| シェアボタン | タップでパルス | CSS Animation |
